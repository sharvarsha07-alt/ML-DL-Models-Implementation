# ML-DL-Models-Implementation
Implementation of 5 Machine Learning and 5 Deep Learning models using Kaggle notebooks.
 # Machine Learning and Deep Learning Models Implementation

## Machine Learning Models
1. Naive Bayes
2. Logistic Regression
3. Support Vector Machine (SVM)
4. Random Forest
5. K-Nearest Neighbors (KNN)

Dataset 1 : SMS Spam Collection Dataset

## Deep Learning Models
1. Artificial Neural Network (ANN)
2. Convolutional Neural Network (CNN)
3. Deep CNN
4. LeNet-5
5. MobileNetV2

Dataset 2 : Dogs vs Cats
